## Project Title
Semantic Satellite Monitoring and Fault Diagnosis Simulator

---

# Project Vision

Build a complete ontology-driven satellite monitoring simulator using:

- Python
- Streamlit
- Owlready2
- RDFLib
- SWRL reasoning

The application must simulate satellite telemetry in real time, inject telemetry into an OWL ontology, execute semantic reasoning, infer faults dynamically, and visualize subsystem health through a professional monitoring dashboard UI.

This is NOT a simple dashboard project.

This is a:
> Semantic Digital Twin + Ontology Reasoning Simulator for Satellite Systems

The ontology file already exists in the root project folder:
`satellite_full.owl`

The ontology contains:
- satellite subsystems
- telemetry concepts
- sensor properties
- fault classes
- SWRL support
- reasoning logic

The application must be modular, scalable, and visually professional.

random

Telemetry values must be injected into ontology individuals dynamically.

satellite.hasTemperature = [92]

Reasoning must happen through ontology inference.
